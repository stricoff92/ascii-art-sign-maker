# ascii-art-sign-maker
Convert images into ASCII art for printing on signs

## Installation

```bash
# create virtual environment
python3 -m venv env
source env/bin/activate

# install deps
pip install -r requirements.txt

```

## usage

```bash

# help message
python3 app.py -h

python3 app.py /path/to/input.jpg 20 93 35

```
